# 6. Сообщения (REST)

Реализованы сообщения: отправка (идемпотентность по clientMessageId, ciphertext base64↔bytea), история с пагинацией по messageId, редактирование и удаление автором, отметка прочтения. Все операции эмитят message.* в outbox; доступ только участнику. Покрыто test/messages.test.ts.
