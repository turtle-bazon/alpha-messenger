# 17. NewChatDialog — точка входа «новый чат»

Синяя «+» в списке чатов стала полноценной точкой входа «новый чат» (как compose в Telegram): кнопка открывает модалку NewChatDialog с двумя вкладками — личный чат по username и группа (название + участники, добавление по username с чипами). Раньше «+» была скрытой кнопкой submit у поля и при пустом поле молча ничего не делала, а групп в клиенте не было вовсе (хотя сервер их уже умел). В клиент добавлен createGroup, HomeScreen — onCreateGroup. Граница центрированного клиента подчёркнута: фон страницы сделан темнее области чата (--page-bg), у .app-shell — боковые рамки и тень (на мобильной ширине убираются). Покрыто e2e/groups.spec.ts; общий хелпер createDirectViaUi (открывает диалог), регресс 9/9 зелёный.
