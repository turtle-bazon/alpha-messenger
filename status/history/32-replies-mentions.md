# 32. Ответы на сообщения и упоминания (@)

Полная реализация системы ответов и упоминаний (#33):

**Сервер:**
- Миграция `reply_to_message_id` в `messages`
- `POST /chats/:id/messages` — поддержка `replyToMessageId`
- `WS message.new` — поле `isReply` в payload
- `loadChat` — поле `unreadMentions`

**Клиент:**
- Кнопка ↩ для ответа на сообщение (hover + контекстное меню)
- Reply preview в пузыре (голубая полоска + клик к оригиналу)
- Плашка ответа над композером
- Бейдж `@N` в ChatList для unread-упоминаний
- `.bubble-reply-to-me` — подсветка красной полоской для сообщений, отвечающих на вас
- Уведомление об ответе: "Ответил(а) на ваше сообщение: ..."
- Навигация по @: кнопка `.conv-mention-nav` внизу чата с пролистыванием
- Свайп вправо для ответа на мобильных (touch-обработка, индикатор ↩)
