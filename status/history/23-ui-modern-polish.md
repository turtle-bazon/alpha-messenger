# 23. Визуальная полировка до современного мессенджера

Визуальная полировка UI до уровня современного мессенджера (known_issues №3), ориентир — десктопный Telegram. Подход: меняли собственные стили, без импорта чужого CSS; у Telegram взяли только эталонные значения (палитра, метрики). Палитра целиком вынесена в CSS-переменные (:root) — единая точка темы. Сайдбар: тень+линия отделяют от области чата, hover/active с переходом, поле поиска с иконкой-лупой и локальной фильтрацией списка, аватары с тенью. Переписка: мягкий градиент+точечный узор фона (чистый CSS), пузыри с радиусом/тенью/градиентом, хвостик у последнего в группе (псевдоэлемент, без SVG), группировка подряд идущих сообщений автора, разделители дат, синий ✓✓. Композер: округлённое поле, круглая кнопка-самолётик с анимацией появления при наличии текста, анимированные точки «печатает». Иконки — единый набор inline-SVG на currentColor (util/icons.tsx); эмодзи не используем из-за разнобоя между ОС. Тёмная тема ([data-theme='dark']) с переключателем в шапке (util/theme.ts: localStorage + дефолт по prefers-color-scheme). Аватары оставлены круглыми (как в Telegram Desktop), вопреки формулировке задачи про «скруглённый квадрат». Покрыто новым e2e ui-polish.spec.ts; регресс e2e 19/19.
