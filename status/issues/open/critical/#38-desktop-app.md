# #38 — Десктопное приложение

Вынести веб-клиент в отдельное десктопное приложение (Electron/Tauri) с иконкой в трее, нативными уведомлениями и автозапуском.

**План реализации:** `work/plans/38-desktop-app.md`

## Требования
- Иконка в трее (сворачивание вместо закрытия)
- Нативные уведомления ОС
- Автозапуск при старте системы (опционально)
- Горячие клавиши (Ctrl+K — поиск, Ctrl+N — новый чат)
- Проверка обновлений
- Работа оффлайн (кэширование)
- Платформы: Windows, macOS, **Linux (обязательно)**
- Сборка бинарников через GitHub Actions

## Технологии
- **Electron** (выбрано)
- Интеграция с существующим Vite-клиентом

## Преимущества
- Лучшая интеграция с ОС
- Стабильная работа
- Нативные уведомления
- Трей для быстрого доступа

## Реализация (в процессе)

### Выполнено
- Создана структура `src/desktop_client/`
- `package.json` с зависимостями (electron, electron-builder)
- `tsconfig.json` для main process
- Main process (`src/main/index.ts`): BrowserWindow, tray, dev/preview modes
- Preload script (`src/main/preload.ts`): безопасный IPC bridge
- Tray module (`src/main/tray.ts`): контекстное меню, двойной клик
- `electron-builder.yml`: конфигурация для Win/Mac/Linux

### Следующие шаги
- Установить зависимости (`npm install`)
- Создать иконки для трея
- Тестовый запуск
- Интеграция с нативными уведомлениями
