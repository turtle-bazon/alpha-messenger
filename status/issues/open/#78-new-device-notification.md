# #78 — Уведомление о входе с нового device ID

## Описание
При входе пользователя с нового device ID (новое устройство/браузер) — отправлять уведомление на все активные сессии пользователя.

## Реализация
- Сервер сравнивает device ID текущего входа с известными для данного user_id
- Если device ID новый — отправляет push/уведомление на все остальные активные сессии: "Вход с нового устройства: <устройство>"
- Уведомление должно содержать информацию об устройстве (тип, IP, время)
- Пользователь должен иметь возможность быстро завершить сессию из уведомления
