# #27 — Статусы онлайн/оффлайн/тайпинг в ChatList

Статусы онлайн/оффлайн/тайпинг в ChatList как в Telegram. Реализовать три состояния аватара:
- **Онлайн**: зелёный кружок внизу справа аватара (12-14px). Только для DM.
- **Оффлайн**: серый кружок внизу справа аватара. Только для DM.
- **Тайпинг**: большой круг-окантовка вокруг всего аватара (2-3px border). Цвет — контрастный с аватаром (пусть клод подберёт подходящий). Появляется: в DM — если собеседник печатает тебе; в групповых чатах — не показывать (или показывать в списке участников при необходимости).
- **Групповые чаты**: кружок статуса (онлайн/оффлайн) НЕ показывать. Тайпинг-окантовку показывать ВЕЗДЕ где есть списки участников в групповых чатах (ChatList, MembersDialog, и т.д.) — если участник печатает в этот чат, обернуть его аватар в тайпинг-окантовку.

Компонент: позиционирование bottom:0 right:0 для кружков, border-radius:50% для тайпинга. Не путать с AvatarStatus (оранжевый «typing» кружок) — это два разных индикатора.
