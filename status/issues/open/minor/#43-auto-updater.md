# #43 — Auto-updater через GitHub Releases

## Описание
Автоматическая проверка и установка обновлений десктопного приложения через GitHub Releases.

## Требования
- Проверка обновлений при запуске приложения
- Уведомление пользователя о доступном обновлении
- Автоматическая скачивание и установка
- Рестарт после установки

## Реализация
- `electron-updater` + GitHub Releases
- Теги релизов как источник версий (v0.1.0, v0.2.0, ...)
- CI/CD (.github/workflows/desktop-build.yml) уже публикует релизы при push тега
- Настройка `publish` в electron-builder.yml (сейчас `false`)
