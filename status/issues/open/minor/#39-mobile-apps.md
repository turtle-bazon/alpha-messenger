# #39 — Мобильные приложения

Нативные мобильные приложения для iOS и Android.

## Требования
- iOS (Swift/SwiftUI)
- Android (Kotlin/Compose)
- Push-уведомления (FCM/APNs)
- Синхронизация с сервером
- Оффлайн-режим

## Приоритет
Низкий — веб-клиент работает в мобильных браузерах, этого достаточно для начала.
