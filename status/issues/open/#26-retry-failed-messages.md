# #26 — Автоповтор неотправленных сообщений

Автоматический повтор неотправленных сообщений. Если отправка не удалась (сеть упала, сервер недоступен) — сообщение помечается как failed и показывается пользователю. При восстановлении связи (или по таймеру) — автоматическая повторная отправка. Порядок строго последовательный: первое неотправленное идёт первым, второе — вторым. Нельзя допустить чтобы более позднее сообщение обогнало раннее. Кнопка «повторить» рядом с failed-сообщением для ручного повтора. При автоматическом повторе — визуальный индикатор (спиннер или анимация).
