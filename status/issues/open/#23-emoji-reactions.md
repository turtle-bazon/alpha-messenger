# #23 — Смайлики: эмодзи в тексте и реакции

Смайлики. Два аспекта: (а) вставка эмодзи в текст сообщения — пикер/панель эмодзи в композере, открытие по кнопке 😊 или по клику, поиск по названию, недавние; (б) реакции на сообщения — возможность добавить эмодзи-реакцию к любому сообщению (как в Telegram/Discord), под сообщением отображаются收集анные реакции с счётчиками, клик по реакции добавляет/убирает свою. Реакции — live через WS, хранятся на сервере.
