# #79 — Настройка уведомлений для каждого чата/канала

## Описание
Пользователь должен иметь возможность настроить уведомления отдельно для каждого чата/канала:
- Включение/отключение звука
- Выбор звука уведомления
- Вибрация
- Приоритет уведомления

## Реализация на Android
На Android каждая комбинация настроек звука — отдельный NotificationChannel. План:
- Канал по умолчанию `alpha_messages` — дефолтный системный звук (для всех чатов без кастомных настроек)
- Кастомные каналы `chat_<chatId>` — для чатов с индивидуальными настройками
- При показе уведомления выбирать канал по chatId

## Реализация на iOS/Web
- iOS: использовать `UNNotificationCategory` и `threadIdentifier` для группировки, звук настраивать через `sound` в payload
- Web: стандартный Web Notification API, звук не настраивается

## Требования
- Настройки хранить в localStorage/IndexedDB (client-side)
- Синхронизация настроек между устройствами (опционально, через server)
- При отключении звука для чата — уведомления тихие (без звука и вибрации)
- При отключении уведомлений для чата — push не отправляется (или отправляется, но без нотификации)
