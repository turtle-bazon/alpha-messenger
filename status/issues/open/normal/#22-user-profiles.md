# #22 — Профили пользователей и заметки

Профили пользователей. Каждый пользователь имеет профиль, который можно открыть (клик по аватару/имени в чате, участнику в списке участников и т.д.). Профиль содержит базовую информацию: username, дату регистрации, последний вход. Помимо этого — каждый пользователь может вести личные пометки о других пользователях (заметки видны только самому автору). Например: «это Петя из отдела маркетинга», «договориться о встрече», «не давать доступ к проекту X». Заметки привязаны к паре (myUserId, targetUserId), редактируемые, с историей изменений.
