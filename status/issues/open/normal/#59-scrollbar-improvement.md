# #59 — Улучшение скроллбаров

Скроллбары выглядят грубо: стрелки по краям, толстая полоса, всегда видны.

## Текущее состояние
- Стрелки по краям (никто не нажимает)
- Толстая полоса, всегда видна
- Стандартный вид браузера/ОС

## Поведение (как в Telegram Desktop)
- Тонкая полоса (3-4px)
- Без стрелок
- Видна только когда курсор мыши в области скролла
- Плавное появление/исчезновение (fade)
- Скроллбар поверх контента, не занимает место

## Реализация
- CSS: `::-webkit-scrollbar` для WebKit/Blink
- `scrollbar-width: thin` + `scrollbar-color` для Firefox
- CSS-анимация opacity при hover на контейнер
- Убрать стрелки через `::-webkit-scrollbar-button { display: none }`
