# #42 — Цветовой badge на иконке вместо числа

## Описание
Вместо отображения количества непрочитанных сообщений на иконке — показывать цветной кружок:
- **Красный** — есть новые сообщения в личные чаты ИЛИ есть ответ на моё сообщение в групповом чате
- **Синий** — есть новые сообщения только в групповые чаты (без личных)
- **Нет кружка** — нет непрочитанных

## Почему не число
Количество нервирует и отвлекает. Достаточно визуального индикатора «есть что-то новое».

## Реализация

### Сервер
- При `message.new` event добавить в payload тип чата и флаг `isReplyToMe` (если это ответ на сообщение пользователя)

### Клиент (Electron)
- Определить тип непрочитанных: есть ли хотя бы один unread в DM или reply-to-me в группе → красный; иначе если есть unread в группах → синий
- Дорисовывать кружок поверх иконки трея через overlay или генерацию новой иконки
- Цвета: красный `#FF3B30`, синий `#2AABEE`

### Иконка
- Базовая иконка `trayIcon.png` (прозрачный фон, синяя α)
- Badge: маленький кружок (8-10px) в правом верхнем углу
- Генерация через `nativeImage` + offscreen canvas или ImageMagick
