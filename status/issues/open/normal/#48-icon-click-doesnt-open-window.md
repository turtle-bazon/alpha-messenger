# #48 — Клик по иконке приложения не открывает окно

## Описание
При клике на иконку приложения (AppImage, .desktop файл) в случае, если приложение уже запущено и свёрнуто в трей, окно не появляется.ичего не происходит.

## Ожидаемое поведение
Клик по иконке должен разворачивать/показывать основное окно приложения (как это работает в Telegram, Discord и т.п.).

## Причина
Electron-приложение должно обрабатывать событие `second-instance` или `activate` (macOS) для повторного показа окна при повторном запуске/клике по иконке. Сейчас это не реализовано.

## Воспроизведение
1. Запустить приложение
2. Свернуть в трей (крестик → свернуть)
3. Кликнуть по иконке приложения в меню/панели
4. Окно не появляется

## Приоритет
normal
