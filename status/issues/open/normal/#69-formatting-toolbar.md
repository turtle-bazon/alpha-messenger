# #69 — Панель форматирования в композере

Пользователям нужно удобно применять markdown-форматирование к тексту. Сделать панель инструментов / контекстное меню при выделении текста в композере.

## Текущее состояние
- Markdown парсится при рендеринге (`renderMarkdown`)
- В композере нет визуальных инструментов форматирования
- Пользователь должен знать синтаксис `**bold**`, `_italic_` и т.д.
- Нет способа вставить ссылку `[текст](url)` кроме ручного набора

## Реализация

### Панель форматирования (как в Telegram Desktop)

#### Расположение
- Над полем ввода (как в Telegram Desktop) — панель с кнопками
- Появляется **только при выделении текста** в композере
- Когда выделение снято — панель скрывается

#### Кнопки (слева направо):
1. **Bold** — `**` + выделенный текст + `**`
2. *Italic* — `_` + выделенный текст + `_`
3. ~~Strikethrough~~ — `~~` + выделенный текст + `~~`
4. `Code` — `` ` `` + выделенный текст + `` ` ``
5. 🔗 Link — диалог ввода URL, оборачивает выделенный текст в `[текст](url)`

#### Поведение
- При выделении текста в композере → панель появляется над полем ввода
- При снятии выделения → панель скрывается
- При клике на кнопку → обёртка выделенного текста markdown-разметкой
- Если текст не выделен (кнопка через горячие клавиши) → вставляет шаблон с плейсхолдером: `**жирный текст**`, `[ссылка](url)` и т.д.
- Курсор остаётся внутри композера

#### Горячие клавиши (опционально, как в Telegram)
- `Ctrl+B` → bold
- `Ctrl+I` → italic
- `Ctrl+K` → link (открывает диалог ввода URL)

### Диалог ввода ссылки
- При нажатии кнопки 🔗 или `Ctrl+K`:
  - Если текст выделен → диалог "Вставьте URL" (текст уже есть)
  - Если текст не выделен → диалог "Введите текст и URL"
- Поля: "Текст ссылки" (prefilled выделенным текстом) и "URL"
- OK → `[текст](url)` вставляется в композер
- Cancel → ничего не происходит

### CSS
- `.formatting-bar` — панель над композером
- `.formatting-btn` — кнопка форматирования (иконка)
- `.formatting-btn.active` — активная кнопка (текст выделен этим форматом)
- `.formatting-btn:hover` — hover-эффект
- Разделитель `|` между группами кнопок

### Интеграция с existing code
- Панель добавляется в `Conversation.tsx` над `.conv-input`
- Обработка выделения текста: `onSelect` на textarea
- Вставка текста: `inputRef.current.setRangeText()` + `inputRef.current.selectionStart/End`

### Тесты
- E2E: выделение текста → клик Bold → текст обёрнут в `**`
- E2E: клик Link → диалог → ввод URL → вставка `[текст](url)`
- E2E: `Ctrl+B` → bold
- E2E: отправка отформатированного сообщения → рендер как bold/italic/etc
