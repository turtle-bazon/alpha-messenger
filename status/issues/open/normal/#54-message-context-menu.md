# #54 — Контекстное меню на сообщениях

Помимо кнопок (удалить, редактировать, ответить) добавить контекстное меню по правому клику на сообщении, как в Telegram Desktop.

## Пункты меню
- Ответить
- Редактировать (только свои)
- Удалить (свои + чужие если владелец группы)
- Копировать текст

## Поведение
- ПКМ на сообщении → контекстное меню рядом с курсором
- Клик вне меню → закрытие
- Выбор пункта → действие + закрытие меню
