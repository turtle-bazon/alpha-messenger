# #50 — Кнопки действий на сообщении наезжают на пузырь

Сейчас кнопки (редактирование, удалить, ответить) появляются рядом с пузырём при hover. Нужно сделать так, чтобы все три кнопки были в одну строку и наезжали на правый верхний угол самого пузыря, выравниваясь по правому краю.

## Визуал
- Три кнопки (✏️ 🗑️ ↩) в одну горизонтальную строку
- Расположены в правом верхнем углу пузыря (наезжают на него)
- Выравнивание по правому краю (последняя кнопка у правого края пузыря)
- Появляются при hover на сообщение
- Для исходящих сообщений (own) — в правом верхнем углу
- Для входящих — аналогично (кнопки одинаковы для всех)
