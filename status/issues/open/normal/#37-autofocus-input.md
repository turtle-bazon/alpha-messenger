# #37 — Автофокус на поле ввода при открытии чата

При любом открытии окна чата (выбор из списка, возврат из настроек, нажатие «назад» на мобильном) — автоматически устанавливать фокус на поле ввода сообщения.

## Поведение
- При выборе чата из ChatList → фокус в поле ввода.
- При возврате из экрана настроек в чат → фокус в поле ввода.
- При нажатии «назад» на мобильном (возврат из переписки к списку) → фокус не нужен (чата нет).
- При переключении между чатами → фокус в поле ввода нового чата.
- При отправке сообщения → фокус остаётся в поле ввода (как сейчас).
- Если поле ввода не видно (чats загружается) → фокусировать после появления.

## Реализация
- В `Conversation` или `HomeScreen` — `useEffect` с зависимостью от `chatId` (или `selectedId`).
- После маунта/рендера — `inputRef.current?.focus()`.
- На мобильных: фокус может открывать клавиатуру — это допустимо (как в Telegram).
- Не фокусировать если пользователь только что закрыл клавиатуру (anti-flicker, ~100ms debounce).

## Тесты
- E2e: выбор чата из списка → фокус в поле ввода (проверка `document.activeElement`).
- E2e: переключение между чатами → фокус в новом чате.
