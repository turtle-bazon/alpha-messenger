# #72 — На мобилке при открытии чата сразу открывается клавиатура

## Проблема
При открытии чата на мобильном устройстве фокус попадает на поле ввода и сразу открывается клавиатура. Это неудобно — пользователь хочет сначала прочитать сообщения.

## Причина
Клавиатура открывается когда `contentEditable` div получает фокус. На мобилках это может происходить при скролле или тапе рядом с полем ввода.

## Решение
- На мобилках не ставить фокус на поле ввода при открытии чата
- Фокус только по прямому тапу на поле ввода
