# #60 — Интеграция иконок Lucide

Заменить кастомные SVG-иконки в `src/web_client/src/util/icons.tsx` на официальную библиотеку Lucide.

## Текущее состояние
Самописные SVG-иконки в `icons.tsx`, написаны под стиль Telegram Desktop. Некоторые могут быть скопированы из Lucide/Feather.

## Цель
- Использовать официальную библиотеку `lucide-react`
- Убрать ручные SVG из `icons.tsx`
- Указать копирайт Lucide (ISC License) в проекте

## Задачи
1. Установить `lucide-react`
2. Заменить импорты в компонентах (Conversation, HomeScreen, ChatList и т.д.)
3. Добавить файл `NOTICE` с копирайтом Lucide в корень проекта
4. Убрать `icons.tsx` (или оставить только кастомные иконки, если Lucide не покрывает все)
5. Проверить, что стили совпадают (strokeWidth, viewBox, size)
6. Экран «О программе» — кнопка в настройках / меню. Информация:
   - Название и версия приложения
   - Копирайт Lucide Icons (ISC License)
   - Ссылка на https://lucide.dev
   - Остальная информация о проекте (автор, репозиторий, лицензия проекта)

## Лицензия
Lucide — ISC License. Совместима с GPL v3. Требуется указание копирайта:
```

ISC License
```
