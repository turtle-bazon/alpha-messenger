# #67 — Экран настроек (как в Telegram)

Создать экран настроек по аналогии с десктопным Telegram. Сейчас настройки уведомлений (колокольчик) и тема (иконка солнца/луны) находятся в шапке — это загромождает интерфейс. Нужно вынести всё в отдельный экран настроек.

## Реализация

### Точка входа
- В шапке слева вместо имени пользователя — кнопка-гамбургер (☰, три горизонтальные полоски). Открывает экран настроек.
- Имя пользователя отображать в заголовке экрана настроек (как в Telegram — «turtle» вверху).

### Экран настроек
- Занимает место списка чатов (левая колонка), как в Telegram.
- Заголовок: «Настройки» с кнопкой «назад» (←) для возврата к списку чатов.
- Список пунктов вертикально, каждый пункт — строка с иконкой слева и текстом:
  1. **Уведомления** — иконка 🔔, подтекст текущего состояния («Вкл»/«Выкл»)
  2. **Тёмная тема** — иконка 🌙/☀️, переключатель (toggle) справа
  3. **О приложении** — иконка ℹ️, пока заглушка

### Раздел «Уведомления»
- При клике — открывается подстраница (внутренняя навигация, не отдельный экран).
- Заголовок: «Уведомления» с кнопкой «назад».
- Пункты:
  - **Звук** — toggle (сейчас `alpha.notif.sound`)
  - **Уведомления браузера** — toggle (сейчас `alpha.notif.browser`). При включении — запрос разрешения (как сейчас в `NotificationSettings.tsx`). При `denied` — toggle заблокирован + подсказка.

### Что убрать из шапки
- Убрать `NotificationSettings` (колокольчик) из шапки.
- Убрать кнопку переключения темы из шапки.
- Кнопка «Выйти» — перенести в экран настроек (внизу, красным текстом, как в Telegram).

### Навигация
- Гамбургер → список чатов заменяется экраном настроек.
- Назад → возврат к списку чатов.
- Уведомления → подстраница уведомлений.
- Назад из подстраницы → обратно к списку настроек.

### Состояние
- `settingsView: 'none' | 'main' | 'notifications'` — текущий вид в левой колонке.
- При `settingsView !== 'none'` — ChatList скрыт, показывается соответствующий экран.

### Стили
- Ориентир — десктопный Telegram: белый фон, разделители между пунктами, hover-эффекты, иконки слева (16-20px), текст справа.
- Toggle-переключатели как в iOS/Telegram — скруглённые, с анимацией.
- Кнопка «Выйти» — внизу списка, красным цветом, без фона.

### Тесты
- E2e: открытие настроек по гамбургеру, переход в уведомления, переключение звука/браузера, возврат назад, выход из аккаунта.

## Решение

Реализовано в коммите `7dec149`. Компонент `SettingsScreen.tsx` с внутренней навигацией (`useState<'main' | 'notifications'>`). Шапка HomeScreen заменена на гамбургер + имя. Удалены `NotificationSettings` и `IconSun/IconMoon` из шапки. Стили: toggle iOS-стиля, красная кнопка «Выйти», разделители. E2e тесты обновлены для новой навигации.
