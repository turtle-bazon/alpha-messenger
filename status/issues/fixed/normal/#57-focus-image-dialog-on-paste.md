# #57 — Фокус на диалоге отправки файла при вставке

При вставке изображения (Ctrl+V / drag-n-drop) диалог отправки файла (ImageEditor) появляется, но фокус остаётся в поле ввода. Нужно автоматически переводить фокус на диалог, чтобы пользователь мог сразу работать с ним (ввести подпись, нажать Enter для отправки).

## Поведение
- При вставке изображения / файл → открытие диалога ImageEditor → автоматический фокус на диалог
- При закрытии диалога → фокус возвращается в поле ввода

## Реализация
- `autoFocus` или `ref.focus()` на корневом элементе ImageEditor при монтировании
- Возврат фокуса в textarea при закрытии диалога
