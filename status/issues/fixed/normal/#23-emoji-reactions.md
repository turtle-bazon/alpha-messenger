# #23 — Смайлики: эмодзи в тексте и реакции

Смайлики. Два аспекта: (а) вставка эмодзи в текст сообщения — пикер/панель эмодзи в композере, открытие по кнопке 😊 или по клику, поиск по названию, недавние; (б) реакции на сообщения — возможность добавить эмодзи-реакцию к любому сообщению (как в Telegram/Discord), под сообщением отображаются собранные реакции с счётчиками, клик по реакции добавляет/убирает свою. Реакции — live через WS, хранятся на сервере.

## Решение

(а) Пикер эмодзи — ранее (EmojiPicker.tsx).

(б) Реакции:
- Сервер: миграция `0007_reactions.sql` (таблица `message_reactions`), PUT `/messages/:id/reactions` (toggle), WS-событие `message.reaction`, loadMessages возвращает reactions.
- Клиент: кнопка 😀 в bubble-actions, мини-пикер из 8 частых эмодзи, отображение реакций под пузырём с счётчиками, клик = toggle, live через WS.
