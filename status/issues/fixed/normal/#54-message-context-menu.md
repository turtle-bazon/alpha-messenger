# #54 — Контекстное меню на сообщениях

Помимо кнопок (удалить, редактировать, ответить) добавить контекстное меню по правому клику на сообщении, как в Telegram Desktop.

## Пункты меню (с иконками, как в Telegram Desktop)
- ↩ Ответить
- ✏️ Редактировать (только свои)
- 🗑️ Удалить (свои + чужие если владелец группы)
- 📋 Копировать текст

## Визуал
- Иконки слева от текста (SVG, как в bubble-actions)
- Отступы, скругления, hover-подсветка
- Разделители между группами пунктов (если есть)

## Поведение
- ПКМ на сообщении → контекстное меню рядом с курсором
- Клик вне меню → закрытие
- Выбор пункта → действие + закрытие меню
