# #55 — Убрать задержку перед подсветкой при переходе

Сейчас при переходе к сообщению (клик по reply preview):
1. Скролл начинается сразу
2. Подсветка начинается через 1.5 сек (фолбэк `setTimeout`)

Если сообщение на той же странице — скролл почти мгновенный, подсветка запаздывает.

## Решение
- Использовать `scrollend` event как основной триггер (уже реализовано)
- Убрать `setTimeout` фолбэк или сильно уменьшить (100–200ms)
- Подсветка должна начинаться сразу после завершения скролла, без лишней задержки

normal
