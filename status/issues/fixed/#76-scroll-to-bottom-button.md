# #76 — Кнопка «к последнему сообщению» при пролистывании вверх

## Описание
Когда пользователь пролистывает ленту сообщений вверх (отошёл от конца), должна появляться плавающая кнопка внизу-справа области чата, которая скроллит к последнему сообщению. Как в Telegram.

## Поведение
- Кнопка скрыта, когда пользователь в конце ленты (в пределах ~100px от дна)
- Кнопка появляется с анимацией при пролистывании вверх
- Клик по кнопке — плавный скролл вниз (`scrollIntoView({ behavior: 'smooth' })`)
- Если есть новые сообщения — показать счётчик «Новых: N» (как в Telegram)
- На мобильных — та же механика

## Контекст
- `Conversation.tsx` — основной скролл-контейнер `.conv-scroll`
- Автоскролл вниз при новом сообщении уже работает
- `#47` и `#59` — были правки автоскролла и скроллбара

## Решение
Реализовано в `Conversation.tsx`:
- `showBackBtn` / `newCount` — стейт видимости кнопки и счётчика
- `onScroll()` — определяет `atBottom` (порог 40px), показывает/скрывает кнопку
- `scrollToBottom()` — плавный скролл вниз
- При получении нового сообщения (`message.new`) если `!atBottomRef.current` — инкремент `newCount`
- При скролле вниз — `newCount` сбрасывается в 0
- CSS: `.scroll-to-bottom` — круглая кнопка внизу-справа с fadeInUp анимацией, `.scroll-to-bottom-badge` — бейдж счётчика
