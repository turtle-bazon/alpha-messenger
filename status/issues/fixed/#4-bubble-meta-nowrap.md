# #4 — Мета-строка пузыря не переносится посимвольно

Время в мета-строке пузыря ломалось посимвольно («20:1 / 3»): у .bubble стоит overflow-wrap: anywhere, что снижает min-content потомков и позволяет браузеру переносить время. Решение: .bubble-meta { white-space: nowrap } — мета (время/статус) всегда в одну строку.
