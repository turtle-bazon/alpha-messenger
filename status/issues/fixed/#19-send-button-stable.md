# #19 — Стабильная кнопка отправки

Кнопка отправки (самолётик) ведёт себя неудобно: когда поле пустое — кнопки нет, справа пустое место; когда начинаешь писать — кнопка появляется. Визуально «прыгает». Нужно: кнопка всегда видна, но в неактивном состоянии (приглушённый цвет, disabled), когда поле пустое. При вводе текста — становится активной (яркий цвет, кликабельная). Без анимации появления/исчезновения — стабильная позиция всегда.

**Решено (задача #19).** Кнопка `.conv-send` всегда в DOM и видна на стабильной позиции; убрана анимация появления (opacity/scale + класс is-visible). При пустом поле — `disabled` с приглушённым фоном/цветом, при наличии текста — активная акцентная. Conversation.tsx, index.css. Покрыто e2e composer.spec.ts.
