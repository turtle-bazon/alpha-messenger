# #10 — Визуальная полировка UI под десктопный Telegram

Визуальная полировка UI до уровня современного мессенджера (ориентир — десктопный Telegram). Подход — меняли свои стили, без импорта чужого CSS; у Telegram взяли только эталонные значения (палитра, метрики). Вся палитра вынесена в CSS-переменные (единая точка темы). Сайдбар: тень+линия отделяют его от области чата, hover/active с переходом, поле поиска с иконкой-лупой и локальной фильтрацией списка по названию, аватары с тенью, badge непрочитанных. Переписка: мягкий градиент+точечный узор фона (чистый CSS), пузыри с радиусом/тенью/градиентом исходящих, хвостик у последнего сообщения группы (треугольник псевдоэлементом, без SVG), группировка подряд идущих сообщений одного автора, разделители дат («Сегодня»/«Вчера»/«24 июня»), синий ✓✓ (прочитано). Композер: округлённое поле со скрепкой, круглая кнопка-самолётик, появляющаяся анимацией только при наличии текста, анимированные точки «печатает». Иконки — единый набор inline-SVG на currentColor (поиск/отправка/скрепка/луна/солнце), эмодзи не используем из-за разнобоя между ОС. Тёмная тема — переключатель в шапке (сохранение в localStorage, дефолт по prefers-color-scheme), палитра по night-теме Telegram. Решение по аватарам: оставлены круглые (как в Telegram Desktop), вопреки буквальной формулировке задачи про «скруглённый квадрат». Покрыто новым e2e ui-polish.spec.ts (тема, поиск, кнопка отправки, разделитель дат); полный e2e-регресс 19/19.
