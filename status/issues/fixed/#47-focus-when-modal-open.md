# #47 — Фокус не смещается на поле ввода при открытом модале

## Описание
Сейчас при открытии модала (например, добавление участников в чат) фокус смещается на поле ввода сообщений. Этого не должно происходить — фокус должен оставаться в модале.

## Причина
Глобальный обработчик фокуса (#40) срабатывает на клик в любом месте, включая область за модалом.

## Решение
Проверять, открыт ли модал (`MembersDialog`, `NewChatDialog`, `MediaViewer` и т.п.), и не устанавливать фокус на input в этом случае.
