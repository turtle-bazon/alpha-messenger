# #18 — Live Draft — реалтайм-превью набора

Live Draft — ультима-фича: реалтайм-превью набираемого сообщения. Когда пользователь начинает набирать текст в поле ввода, собеседник (или участники группы) видит его облачко в реальном времени — буквы появляются по мере набора.

## Реализация ✅

### Клиент-отправитель
- При вводе текста (debounce ~2s throttle + 1s flush) WS-событие `typing` с текущим `draft`
- При очистке поля — `typing` с пустым draft
- При отправке сообщения — draft автоматически удаляется

### Клиент-получатель
- По событию `typing` с непустым draft — облачко-превью в области переписки
- Облачко обновляется в реальном времени
- Когда пользователь прекращает набор — облачко исчезает (fade-out + slide-down)
- Не вызывает уведомления (нет звука, badge, push)

### Визуальное отличие
- Пунктирная рамка (`border-style: dashed`)
- Полупрозрачный фон (`opacity: 0.7`)
- Подпись «печатает...» вместо времени
- Анимация появления (`draft-in` keyframes)

### Сервер
- WS-событие `typing` с полем `draft` (string, опционально)
- Хранение draft — не требуется (эфемерная передача через WS)
- Группы: draft-облачко видно всем участникам, кроме автора

### Файлы
- `src/server/src/ws.ts`: поле `draft` в typing event
- `src/web_client/src/chats/Conversation.tsx`: draft preview bubble, `hasDraft`, autoscroll
- `src/web_client/src/chats/useTyping.ts`: `Map<userId, draft>`
- `src/web_client/src/index.css`: `.bubble-draft` стили, `draft-in` анимация
