# #8 — Участники группы — из знакомых пользователей

Создание группы: участников выбирают из знакомых пользователей (собеседники существующих личных чатов), а не свободным вводом username (нельзя добавить того, с кем ещё нет переписки). ChatList собирает уникальных участников direct-чатов (исключая себя — создатель входит в группу автоматически на сервере) и отдаёт в NewChatDialog; вкладка «Новая группа» показывает поиск + список кандидатов с чипами выбранных, у пользователя без чатов — подсказка. Группа без участников по-прежнему не создаётся. Покрыто e2e groups.spec.ts; helper createGroupViaUi в members.spec.ts переведён на новый флоу.
