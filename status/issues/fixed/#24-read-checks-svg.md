# #24 — Красивые галочки прочтения (SVG)

Галочки прочтения (✓✓) — сделать посимпатичнее. Сейчас выглядят как простые текстовые символы Unicode. Ориентир — Telegram: маленькие (12-14px), аккуратные, с правильным отступом от времени. Реализовать через inline SVG (двойная галочка как путь path), а не через текстовые символы. Два размера галочки: одна (отправлено) и двойная (доставлено/прочитано). Три статуса: отправлено (одна серая), доставлено (две серые), прочитано (две синие/зелёные). Отступ между временем и галочками — 2-4px. Сама галочка — 14x14px, stroke-width ~1.5, без заливки (только контур).

**Решено (задача #24).** Текстовые ✓/✓✓ заменены на inline-SVG (IconCheck/IconChecks, тонкий контур ~1.5px, без заливки, currentColor). Статусы своих сообщений: отправка — спиннер (.bubble-spinner), ошибка — «!», отправлено — одиночная серая галочка, прочитано — двойная синяя (var(--check-read)). Добавлен data-status (sending|failed|sent|read) для семантических проверок. Замечание: отдельного «доставлено» (две серые) нет — сервер не различает доставку от отправки, маппинг честный по нашим реальным состояниям. icons.tsx, Conversation.tsx, index.css. Покрыто e2e composer.spec.ts (interactions/two-clients переведены на data-status).
