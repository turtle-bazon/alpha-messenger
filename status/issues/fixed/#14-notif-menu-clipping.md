# #14 — Меню колокольчика обрезалось слева

[Баг] Выпадающее меню настроек уведомлений (колокольчик в шапке) обрезалось слева, текст был нечитаем («Звук» → «вук»). Причина: колокольчик — не самая правая кнопка шапки (за ним тема и «Выйти»), его правый край ~211px от левого края; меню фиксированной ширины (min-width 240px), выровненное по правому краю кнопки (right: 0), уходило влево до ~-28px и обрезалось левой границей .app-shell (overflow: hidden). При ширине окна 1280 .app-shell занимает весь вьюпорт, поэтому буквальное «выравнивание по правому краю кнопки» для меню 240px геометрически невозможно — оно неизбежно вылезает за левый край. Решение как в Telegram: меню переведено на position: fixed (вырывается из overflow-контейнера), а координаты top/left считает компонент (NotificationSettings, useLayoutEffect по rect кнопки) с клампом по вьюпорту — выравнивание по правому краю кнопки с поджатием влево, чтобы меню целиком помещалось на экране. Покрыто e2e notifications.spec.ts (меню в пределах вьюпорта, обе строки видимы).
