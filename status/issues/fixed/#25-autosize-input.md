# #25 — Авторасширение поля ввода сообщения

Поле ввода сообщения должно расширяться по мере ввода текста. Сейчас при многострочном тексте окно скроллится и предыдущий текст не виден. Нужно: textarea растёт в высоту по мере добавления строк (макс. высота ~150-200px, после — внутренний скролл), отправка по Enter (Shift+Enter — перенос строки). Рост плавный, без «прыжков».

**Решено (задача #25).** Поле ввода — textarea с авторасширением: высота сбрасывается в auto и подгоняется под scrollHeight до потолка 160px (useLayoutEffect по input), дальше внутренний скролл. Enter отправляет, Shift+Enter — перенос строки (onKeyDown, isComposing отсекает IME). Conversation.tsx, index.css (.conv-textarea). Покрыто e2e composer.spec.ts.
