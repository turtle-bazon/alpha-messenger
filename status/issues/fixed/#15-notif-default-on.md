# #15 — Галочка уведомлений включена по умолчанию

Настройки уведомлений: галочка браузерных уведомлений по умолчанию ВКЛЮЧЕНА, и приложение само запрашивает системное разрешение при загрузке (если включено и разрешение ещё 'default'), повторно не спрашивает при уже выданном/запрещённом. Фактически реализовано вместе с механизмом уведомлений (см. п.13): дефолт «включено» через readFlag (отсутствие ключа в localStorage = вкл), ensureBrowserPermission в HomeScreen на маунте проверяет настройку и Notification.permission. Задача оставалась в known_issues по недосмотру — перенесена без доработки кода. Покрыто e2e notifications.spec.ts («приложение само запрашивает разрешение на уведомления при входе»).
