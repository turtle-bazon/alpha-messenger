# #7 — Глазик показа/скрытия пароля

На экранах входа и регистрации добавлен глазик показа/скрытия пароля (как в Telegram): общий компонент PasswordInput чередует type input password/text по клику на иконку (inline SVG eye/eye-off). Подпись кнопки без слова «пароль», чтобы getByLabel('Пароль') в тестах не цеплял её вместе с полем. Покрыто e2e auth.spec.ts.
