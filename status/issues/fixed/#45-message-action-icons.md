# #45 — Иконки редактирования/удаления в углу сообщения

## Описание
Сейчас при наведении на сообщение появляются текстовые ссылки «редактировать», «удалить». Нужно заменить их на иконки в правом верхнем углу пузыря сообщения (как в Telegram Desktop).

## Текущее поведение
- Текстовые кнопки «редактировать», «удалить» появляются при hover
- Занимают место, выглядят неестественно

## Реализация
- Контейнер иконок в `position: absolute; top: 4px; right: 4px` внутри `.bubble`
- Иконки: карандаш (edit), корзина (delete), стрелка влево (reply), стрелка вправо (forward)
- Появляются при hover на сообщение
- Для своих сообщений — edit + delete + reply + forward
- Для чужих сообщений — reply + forward
