# #73 — Уведомления о реакциях

## Описание
Пользователь никак не узнаёт, что на его сообщение поставили реакцию. Может только догадаться, увидев значок в UI. Нужно уведомлять об этом — push-уведомлением (на мобильном/десктопе) и/или внутри приложения.

## Решение
- Клиент: обработчик `message.reaction` в HomeScreen.tsx — browser notification + звук
- Функция `notifyReaction()` в notifications.ts (аналог `notifyIncoming`)
- Уведомление только при `action === 'added'` (не при удалении)
- Свои реакции не уведомляем (`p.userId === myId` → skip)
- Уведомление только когда вкладка не активна (`inForeground()` → return)
- Имя реактора берётся из `chat.participants`
- Формат: "Username 👍" в title чата
- Серверная часть push через outbox уже работает (архитектура есть, стаб)
