# Tray Icon

Place `trayIcon.png` here (16x16 or 22x22 for Linux, 22x22 for Windows).

For macOS, use `trayIconTemplate.png` (black on transparent, macOS will auto-invert).

## Requirements

- Format: PNG with transparency
- Size: 16x16 (macOS), 22x22 (Windows/Linux)
- Style: Simple, recognizable at small sizes
- Color: Single color (white/black) works best for system trays
