Модель

## Устройство

Устройство пользователя - сущность с deviceId: UUID. На устройстве пользователя будет храниться ключ шифрования данного устройства, а также ключ учетной записи. Ключ устройства нужен для организации собственного транспортного слоя шифрования поверх используемых протоколов.

## Учетная запись (пользователь)

Учетная запись пользователя - сущность с username, выбираемым пользователем. username не меняется после создания учетной записи.
Учетная запись имеет ключ (не путать с ключом устройства).

## Группа

Чат для совместного общения нескольких пользователей.
