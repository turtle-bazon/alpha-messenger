# UI

## Образец — десктопный Telegram

В реализации интерфейса ориентируемся на десктопную версию Telegram (см. также CLAUDE.md, раздел «Образец»). Это касается интерфейса и поведения; вся остальная реализация — полностью своя.

## CSS-стили

При создании CSS-стилей нужно ориентироваться на похожесть на Telegram: общая раскладка (список чатов слева, область переписки справа), вид пузырей сообщений, отступы, типографика, цветовые акценты, ховер- и активные состояния. Если для конкретной фичи не указано иное, визуально делаем «как в Telegram».
